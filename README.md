# Aura Chique - Frontend

Repositório contendo a interface web do e-commerce Aura Chique, construída nativamente com HTML, CSS e JavaScript. 

A aplicação opera sob uma arquitetura de acesso controlado. A partir do mesmo ecossistema de código, a interface renderiza a jornada de compra (storefront) otimizada para conversão ou o painel de back-office restrito, onde administradores gerenciam o catálogo de produtos, controle de inventário e fluxo logístico de pedidos.
